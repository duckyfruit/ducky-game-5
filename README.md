# Hide and Quack

Author: Taylor Kynard

Design: find your friend before the time runs out!

Screen Shot:

![Screen Shot](screenshot.png)

How To Play:

WASD is used to move your character, and the mouse is used to look around. Run into the other duck to
"find" them!
